Completed integration test infrastructure setup.

**What was done:**
- Copied all 4 Python fixture files (basic.md, escaping.md, plain.md, stripped.md) into `apps/cli/testdata/`
- Created `internal/testutil` package with:
  - `StartServer(t, extra)` — copies fixtures + extra files into a temp dir, starts a real HTTP server on 127.0.0.1:0, returns `*Server` with URL, TempDir, and Close()
  - `Get(t, baseURL, path)` — fetches URL and returns body as string
  - `GetResponse(t, baseURL, path)` — fetches URL and returns full `*http.Response`
  - `TestdataDir()` — resolves testdata/ path via `runtime.Caller`
- Added 4 infrastructure tests verifying fixture copying, extra files, server responses, and Get helper
- All tests pass (`make test` green)

**Design decisions:**
- Used `runtime.Caller` to locate testdata/ relative to the source file, avoiding brittle working-directory assumptions
- Server binds to `127.0.0.1:0` for random port allocation
- `StartServer` accepts `map[string]string` for extra files, allowing tests to add custom content without modifying shared fixtures
